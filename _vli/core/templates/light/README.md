# Template Light

Template editorial claro para landing pages de imoveis de alto padrao.

## Direcao visual
- Newsreader italic nos titulos
- fundo claro com textura visual leve
- botoes e acentos em tom metalico
- bordas retas, espacamento generoso e composicao minimalista

## Estrutura
- `index.html`: layout completo do template
- `css/variables.css`: paleta ativa
- `css/palettes/*.css`: variacoes prontas
- `js/config-loader.js`: integra com `painel.html`
- `js/main.js`: abas de plantas e formulario/WhatsApp

## Observacoes
- O template funciona sozinho com os dados demo do HTML.
- Quando aberto via `?lp=<id>`, ele aplica os dados salvos no painel.
- A secao de localizacao do painel controla `#bairro` e `#localizacao` ao mesmo tempo neste template.
